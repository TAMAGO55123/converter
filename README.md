# Converter
![昆布](ico.png)

## 以下の機能を搭載しています。
- 動画・音声ダウンロード

  ダウンロード可能なサイト一覧は以下のリンクから

  https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md

  主要なサイト

  - Youtube
  - ニコニコ動画
  - SoundCloud

- WAV(PCM) へ変換

  MMDで読み込めるwavに変換します
